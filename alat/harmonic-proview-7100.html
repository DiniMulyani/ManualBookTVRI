1. Peringatan Keselamatan Kerja (K3)

PERHATIAN! KUALITAS SINYAL & KARTU LISENSI
  • Pastikan konektor F (L-Band dari LNB/Parabola) terpasang kencang. Konektor kendor sedikit saja bisa menyebabkan sinyal 'No Lock'.
  • Jangan mengubah konfigurasi IP Management kecuali diinstruksikan. IP Conflict akan memutus akses ke alat.
  • Fitur dekoding (misal: H.264, 4:2:2) mungkin bergantung pada lisensi. Jangan me-reset pabrik alat tanpa backup lisensi.

2. Prosedur Koneksi Awal

  1. Hubungkan kabel L-Band (dari LNB Parabola) ke port 'RF/L-Band In'.
  2. Hubungkan kabel LAN (RJ45) dari port 'Management' ke Jaringan LAN/Laptop.
  3. Hubungkan kabel output video (misal: 'SDI Out 1') ke MCR atau Multiviewer.
  4. Hubungkan kabel power dan nyalakan IRD. Tunggu hingga proses booting selesai (bisa 1-2 menit).

3. Prosedur Konfigurasi (WebGUI)

IP Address default pabrikan untuk Harmonic ProView 7100 bisa bervariasi, namun umumnya `192.168.1.200` atau didapat via DHCP. Periksa di panel depan (jika ada) atau hubungi teknisi jaringan.

  1. Pastikan IP Laptop/PC Anda satu segmen dengan IRD (misal: 192.168.1.10).
  2. Buka Browser dan ketik IP Address IRD (misal: `http://192.168.1.200`).
  3. Masuk menggunakan username dan password (misal: admin/admin atau admin/proview).
  4. Konfigurasi Input (Satelit):
    • Masuk ke menu `Configuration -> Input -> RF Input`.
    • Atur 'Input Type' ke 'DVB-S' atau 'DVB-S2'.
    • Masukkan 'Frequency' (Frekuensi L-Band, misal: 1550 MHz).
    • Masukkan 'Symbol Rate' (misal: 30000 Ksym/s).
    • Atur 'FEC' (misal: 3/4 atau Auto).
    • Atur 'LNB Power' ke 'On' (13V/18V) jika IRD memberi daya langsung ke LNB.
  5. Cek Status Lock:
    • Masuk ke `Status -> Input`.
    • Periksa 'RF Lock Status' (Harus 'Locked' - Hijau).
    • Periksa 'TS Lock Status' (Harus 'Locked' - Hijau).
  6. Konfigurasi Service (Program):
    • Masuk ke `Configuration -> Services`. Alat akan men-scan Transport Stream.
    • Cari Service Name yang benar (misal: 'TVRI NASIONAL') dari daftar.
    • Pilih Service tersebut dan pastikan 'Output' ter-routing ke dekoder (misal: 'Decoder 1').
  7. Konfigurasi Output Video:
    • Masuk ke `Configuration -> Output -> SDI Output`.
    • Atur 'Output Format' (misal: 1080i/50) agar sesuai dengan standar MCR.
  8. Klik 'Apply' atau 'Save' untuk menyimpan konfigurasi.

5. Troubleshooting

Apa masalah Anda?

Sudah periksa parameter input (Frekuensi, Symbol Rate) di WebGUI?

Ini berarti sinyal fisik satelit ada, tapi data Transport Stream tidak terbaca.

Sinyal & TS Lock OK, tapi output hitam. Sudah cek pemilihan Service/Program?

Solusi: Cek Fisik Parabola/LNB

Parameter sudah benar tapi sinyal fisik tidak ada. Masalah ada di luar:

  1. Periksa kabel L-Band dari parabola ke IRD (pastikan tidak putus/konektor kendor).
  2. Periksa LNB di parabola (mungkin mati/rusak).
  3. Periksa arah/pointing parabola (mungkin bergeser).

Solusi: Cek Lisensi / Output

Sinyal terkunci dan Service benar, tapi output hitam. Kemungkinan:

  1. Sinyal ter-enkripsi (Scrambled) dan IRD tidak punya lisensi BISS/CAS.
  2. Cek menu `Configuration -> Output -> SDI Output` pastikan sudah 'Enabled'.

Solusi: Pilih Service yang Benar

Masuk ke `Configuration -> Services`. TS Lock OK berarti IRD sudah membaca MUX. Pastikan Anda memilih Service Name yang benar (bukan 'Data' atau 'Radio') dan me-routingnya ke 'Decoder 1'.
