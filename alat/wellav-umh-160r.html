1. Peringatan Keselamatan Kerja (K3)

PERHATIAN! KUALITAS SINYAL & IP CONFLICT
  • IP default alat ini adalah 192.168.1.188. Pastikan IP Laptop Anda satu segmen (misal: 192.168.1.10) sebelum konfigurasi.
  • Pastikan konektor F (L-Band dari LNB) terpasang kencang. Konektor kendor sedikit saja bisa menyebabkan sinyal 'Unlock'.
  • Selalu klik 'Apply' setelah mengubah pengaturan di WebGUI.

2. Prosedur Koneksi Awal

  1. Hubungkan kabel L-Band (dari LNB Parabola) ke port 'Tuner In' atau 'RF In'.
  2. Hubungkan kabel LAN (RJ45) dari port 'ETH' atau 'Management' ke Jaringan LAN/Laptop.
  3. Hubungkan kabel output video (misal: 'SDI Out 1') ke MCR atau Multiviewer.
  4. Hubungkan kabel power dan nyalakan IRD. Tunggu hingga proses booting selesai.

3. Prosedur Konfigurasi (WebGUI)

IP Address default pabrikan untuk Wellav UMH 160R adalah 192.168.1.188.

  1. Atur IP Address Laptop/PC Anda ke segmen yang sama (misal: 192.168.1.10, Subnet 255.255.255.0).
  2. Buka Browser dan ketik IP default: `http://192.168.1.188`.
  3. Masuk menggunakan username dan password default (misal: admin/123456).
  4. Konfigurasi Input (Satelit):
    • Masuk ke menu `Input -> Tuner`.
    • Pilih Tuner 1. Atur 'Tuner Type' ke 'DVB-S' atau 'DVB-S2'.
    • Masukkan 'Frequency' (Frekuensi L-Band, misal: 1550).
    • Masukkan 'Symbol Rate' (misal: 30000).
    • Atur 'Polarity' (Horizontal/Vertikal) dan 'LNB Power' (13V/18V) sesuai kebutuhan LNB Anda.
    • Klik 'Apply'.
  5. Cek Status Lock:
    • Masuk ke `Status -> Tuner`.
    • Periksa 'Status' (Harus 'Lock' - Hijau).
    • Periksa 'C/N Ratio' (Harus memiliki nilai yang baik, misal: >10 dB).
  6. Konfigurasi Service (Program):
    • Masuk ke `Input -> TS Setting`. Klik 'Search'. Alat akan men-scan Transport Stream.
    • Masuk ke `Output -> Decoder 1`.
    • Pada 'Program Select', pilih Service Name yang benar (misal: 'TVRI NASIONAL') dari daftar program yang sudah di-scan.
    • Pastikan 'Video PID' dan 'Audio PID' terisi otomatis.
  7. Konfigurasi Output Video:
    • Masuk ke `Output -> A/V Output`.
    • Atur 'Video Output' ke 'SDI'.
    • Atur 'Video Format' (misal: 1080i_50Hz) agar sesuai dengan standar MCR.
    • Klik 'Apply'.
  8. Periksa output video di monitor SDI Anda.

5. Troubleshooting

Apa masalah Anda?

Sudah periksa parameter input (Frekuensi, Symbol Rate, Polarity) di WebGUI?

Sudah cek `Output -> Decoder 1` dan memilih Program/Service?

Solusi: Cek Fisik Parabola/LNB

Parameter sudah benar tapi sinyal fisik tidak ada. Masalah ada di luar:

  1. Periksa kabel L-Band dari parabola ke IRD (pastikan tidak putus/konektor kendor).
  2. Periksa LNB di parabola (mungkin mati/rusak).
  3. Periksa arah/pointing parabola (mungkin bergeser).

Solusi: Periksa Ulang Parameter

Pastikan Frekuensi, Symbol Rate, Polarity, dan FEC sudah 100% sesuai dengan data dari provider satelit. Kesalahan sedikit saja akan menyebabkan 'Unlock'.

Solusi: Cek Enkripsi (Scrambled)

Sinyal terkunci dan Service benar, tapi output hitam. Kemungkinan:

  1. Sinyal ter-enkripsi (Scrambled). Cek `Status` apakah ada 'Scrambled' warning.
  2. Jika Ya, Anda perlu modul CAM dan Smartcard BISS/CAS yang sesuai.
  3. Jika Tidak, cek `Output -> A/V Output` pastikan Video Format (1080i/50) sudah benar.

Solusi: Pilih Service yang Benar

Masuk ke `Output -> Decoder 1 -> Program Select`. Pastikan Anda memilih Service Name yang benar (bukan 'Data' atau 'Radio') dan klik 'Apply'.
