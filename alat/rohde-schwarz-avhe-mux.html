1. Peringatan Keselamatan Kerja (K3)

PERHATIAN! KONFIGURASI KRITIS & HEADEND
  • Platform AVHE adalah sistem headend terintegrasi. Perubahan pada MUX dapat mempengaruhi fungsi lain (Encoder, Scrambler).
  • Pastikan IP Management unik. Gunakan username/password yang kuat.
  • Selalu 'Activate' konfigurasi setelah melakukan perubahan agar diterapkan.

2. Prosedur Koneksi Awal

  1. Hubungkan kabel input stream (ASI atau IP) dari Encoder ke port input pada AVHE.
  2. Hubungkan kabel output (ASI atau IP) dari port output AVHE ke Pemancar.
  3. Hubungkan kabel LAN (RJ45) dari port 'Management' AVHE ke Jaringan LAN/Laptop.
  4. Nyalakan AVHE. Tunggu hingga sistem siap (bisa beberapa menit).

3. Prosedur Konfigurasi (WebGUI)

IP Address platform AVHE harus diketahui (Contoh: `192.168.1.12`).

  1. Atur IP Address Laptop/PC Anda ke segmen yang sama.
  2. Buka Browser dan ketik IP Address AVHE (misal: `http://192.168.1.12`).
  3. Masuk menggunakan username dan password.
  4. Konfigurasi Input Stream:
    • Navigasi ke `Configuration -> Inputs`.
    • Tambahkan input baru (misal: ASI Input). Konfigurasikan port fisik.
    • Pastikan status input 'OK'.
  5. Membuat Multiplex (Output TS):
    • Navigasi ke `Configuration -> Multiplexing -> Multiplexes`.
    • Klik 'Add' untuk membuat Multiplex baru. Beri nama dan atur TS ID, Network ID.
  6. Menambahkan Services ke Multiplex:
    • Pilih Multiplex yang baru dibuat. Masuk ke tab 'Services'.
    • Klik 'Add Service'.
    • Pilih 'Input Stream' sumber.
    • Pilih Service yang ingin dimasukkan dari daftar.
    • Atur PID Output jika perlu di-remap.
    • Ulangi untuk Service lain.
  7. Konfigurasi PSI/SI:
    • Masuk ke tab 'PSI/SI' di dalam konfigurasi Multiplex.
    • Aktifkan PAT, PMT, SDT, NIT. Masukkan informasi yang diperlukan (Service Name, Provider, Network Name).
  8. Konfigurasi Output Fisik:
    • Navigasi ke `Configuration -> Outputs`.
    • Tambahkan output baru (misal: ASI Output).
    • Pilih port fisik.
    • Pilih 'Source' = Multiplex yang tadi dibuat.
    • Atur 'Bitrate' dan mode (CBR/VBR).
  9. Klik 'Activate Configuration' di pojok atas untuk menerapkan semua perubahan.

5. Troubleshooting

Apa masalah Anda?

Solusi: Cek Output Config & Activate

Pastikan `Configuration -> Outputs` sudah benar memilih 'Source' Multiplex. Periksa juga Bitrate. Yang paling penting: Apakah Anda sudah klik 'Activate Configuration'?

Solusi: Cek Input & Service Mapping

Pastikan Input Stream sumber service statusnya OK. Lalu cek di `Configuration -> Multiplexing -> Multiplexes -> [Nama MUX] -> Services`, pastikan service tersebut sudah ditambahkan.

Solusi: Cek Konfigurasi PSI/SI

Masuk ke `Configuration -> Multiplexing -> Multiplexes -> [Nama MUX] -> PSI/SI`. Pastikan tabel PAT dan PMT aktif dan semua PID service sudah tercantum. Cek juga interval.
