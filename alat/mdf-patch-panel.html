2. Prosedur Standar Patching

  1. Verifikasi Port Input dan Port Tujuan yang dibutuhkan (lihat diagram alir).
  2. Ambil kabel patch cord dengan panjang yang sesuai (jangan terlalu panjang/pendek).
  3. Hubungkan Patch Cord: Port Sumber -> Port Tujuan di Patch Panel.
  4. Catat perubahan koneksi di logbook atau sistem dokumentasi digital.
  5. Uji sinyal (continuity test) untuk memastikan koneksi berhasil.

3. Wizard Troubleshooting: No Signal Setelah Patching

Gunakan panduan ini jika sinyal hilang setelah melakukan patching.

Langkah 1: Apakah koneksi Patch Cord longgar di salah satu ujung?

Langkah 2: Apakah Anda yakin Port Sumber di Patch Panel memiliki sinyal?

Langkah 3: Apakah Anda menggunakan Patch Cord yang benar (misalnya, BNC untuk video, RJ45 untuk data)?

Solusi: Koneksi Longgar

Dorong atau kunci kembali konektor Patch Cord hingga terasa pas. Putar konektor BNC searah jarum jam hingga terkunci.

Solusi: Periksa Sinyal Sumber

Sinyal hilang sebelum MDF. Cek alat sumber (misalnya, kamera, VTR) apakah sudah menyala dan mengeluarkan sinyal normal.

Solusi: Kabel Tidak Sesuai

Pastikan Anda menggunakan tipe kabel yang sesuai (misalnya, kabel SDI 75-Ohm untuk Video, bukan kabel 50-Ohm). Ganti Patch Cord Anda.

Solusi: Kegagalan Port Patch Panel

Jika semua langkah di atas sudah diperiksa, kemungkinan port di Patch Panel rusak. Coba gunakan port sebelahnya. Jika berhasil, segera laporkan port yang rusak untuk perbaikan.
